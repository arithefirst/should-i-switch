# Should I Switch

A simple website to help users decide wether or not switching to Linux is a viable choice for them  

## Tech Stack

- Next.js 15 (App Router)  
- React 19 + TypeScript  
- Tailwind CSS for styling  

## Getting Started

### Prerequisites

- Node.js ≥ 18.x  
- npm or yarn  

### Installation

1. Clone the repo  
   ```bash
   git clone https://github.com/arithefirst/should-i-switch.git
   cd should-i-switch
   ```
2. Install dependencies  
   ```bash
   npm install
   # or
   yarn
   ```
3. Run the development server  
   ```bash
   npm run dev
   # or
   yarn dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` — start Next.js in development mode  
- `npm run build` — build for production  
- `npm run start` — start the production server  
- `npm run lint` — run ESLint  

## Project Structure

- `/src/app` — Next.js pages & layout  
- `/src/components` — reusable UI components  
- `/src/questions.ts` — quiz data & scoring logic  
- `/public` — static assets (logos, images)
